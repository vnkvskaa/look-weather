# look.

Phone-first дневник луков × погоды.

## Что умеет

- Профиль погоды на сегодня или выбранный день (feels-like, ветер, влажность, осадки, облачность)
- Рекомендации луков под эту погоду
- Лог лука с фото + фидбек «холодно / норм / жарко»
- Локальное хранение на телефоне + экспорт/импорт в Файлы / iCloud Drive
- Запасная копия луков в аккаунте GitHub (ключ только на телефоне; публичный репозиторий — код)

## Прод

Постоянный URL (ноут не нужен):

**https://vnkvskaa.github.io/look-weather/**

На iPhone: Safari → Поделиться → На экран «Домой».

Деплой сам уходит на GitHub Pages при пуше в `main`.

### Запасная копия на GitHub

В настройках приложения — пошаговая инструкция «как сделать запасную копию». Коротко: бесплатный аккаунт GitHub → ключ на [странице токенов](https://github.com/settings/tokens/new?scopes=repo&description=look) (нужна галочка **`repo`**, не только `gist`) → вставить ключ в look. → сохранить первую копию. Приложение само создаёт закрытый репозиторий `look-weather-data` (луки + сжатые фото). Ключ хранится только на телефоне; публичный репозиторий look-weather — это код, не данные.

## Локальная разработка (по желанию)

```bash
npm install
npm run dev
```

Локальный `vite` нужен только пока правишь код — для повседневного пользования хватает прод-URL.
